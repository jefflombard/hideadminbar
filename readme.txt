== Description ==

Simply activate the plugin to remove the admin bar for non-admin users.
To see the dashboard as a user that cannot see the admin bar go to (http://yoursite.com/wp-admin)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `simple_hide_admin_bar.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
